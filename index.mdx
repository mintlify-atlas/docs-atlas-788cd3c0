<Note>
Skyline enables runtime function hooking, dynamic plugin loading, and code patching for Nintendo Switch games, with primary support for Super Smash Bros Ultimate.
</Note>

## Key Features

Skyline provides a comprehensive framework for modifying game behavior at runtime.

## Why Skyline?

Skyline is derived from OdysseyReversed and Starlight, providing a battle-tested foundation for game modding:

- **Production-Ready**: Used by thousands of modders worldwide
- **Extensible**: Plugin architecture allows unlimited customization
- **Well-Documented**: Comprehensive API documentation and examples
- **Active Community**: Multiple game-specific forks and derivates available

<Info>
While Skyline is optimized for Super Smash Bros Ultimate, it has been adapted for many other Nintendo Switch games including Animal Crossing, Pokémon Sword/Shield, Xenoblade Chronicles, and more.
</Info>
